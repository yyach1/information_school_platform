# 学院学生综合服务与党团管理平台 — 使用手册

## 一、平台入口

| 用户 | 使用方式 | 地址 |
|------|---------|------|
| 教师/管理员 | 电脑浏览器 | `http://10.10.0.30/admin/login` |
| 学生 | 微信小程序（测试阶段） | 微信开发者工具导入源码运行 |

> 小程序暂未上线微信正式版，测试阶段通过下载源码 + 微信开发者工具使用。详见下方说明。

---

## 二、小程序测试方法（学生端）

### 2.1 准备工具

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 从 GitHub 下载小程序源码：

```bash
# 克隆项目到本地
git clone https://github.com/yyach1/information_school_platform.git
```

源码位于 `miniprogram/` 目录下。

### 2.2 导入项目

1. 打开微信开发者工具 → 点击「**导入**」
2. 选择 `miniprogram/` 目录
3. AppID 选择「**测试号**」或使用已有 AppID
4. 点击确定导入

### 2.3 配置设置

1. 点击右上角「**详情**」→「**本地设置**」
2. 勾选「**不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书**」
3. 确认 `app.js` 中 `baseUrl` 指向正确的后端地址：
   ```js
   baseUrl: 'http://10.10.0.30/api',
   ```
   如果后端在本机开发环境，改为：
   ```js
   baseUrl: 'http://localhost:8080/api',
   ```

### 2.4 开始测试

1. 点击工具栏「**编译**」（或 Ctrl + B）
2. 登录页面输入学生账号密码
3. 即可使用所有小程序功能

### 2.5 真机预览

1. 点击工具栏「**预览**」
2. 手机微信扫码，可在手机上测试实际效果
3. 注意：手机需与服务器在同一网络（或服务器外网可访问）

## 三、学生使用指南（小程序）

### 3.1 登录

打开小程序 → 输入学号和密码 → 点击登录

> 首次使用请联系管理员创建账号。忘记密码请联系辅导员重置。

### 3.2 首页

登录后进入首页，顶部显示问候语和你的姓名。下方可看到：

- **我的党团进度** — 当前可办理的流程，点击进入办理
- **快捷服务** — 党团进度、材料上传、证明办理、智能问答

### 3.3 办理党团流程

1. 点击底部「**党团进度**」Tab
2. 看到可办理的流程（如「入党申请流程」「入团申请流程」）
3. 点击「进入办理」开始流程
4. 进入流程详情页，可看到：
   - 当前所在节点
   - 进度时间轴
   - 已提交的材料及审核状态

### 3.4 上传材料

1. 在流程详情页，点击「**提交材料**」按钮
2. 页面显示当前节点需要上传的材料清单
3. 每个材料项会标注：
   - 是否必传（红色 *）
   - 文件大小限制
   - 支持的文件格式
4. 点击「+ 选择文件」→ 选择文件 → 自动上传
5. 所有必传材料上传完成后，点击「**提交审核**」

### 3.5 申请电子证明

1. 点击底部「**我的证明**」Tab
2. 点击右上角「申请证明」
3. 选择证明类型：
   - 在学证明、请假申请、盖章申请、党员证明、成绩单证明
4. 填写申请信息，上传附件（如需要）
5. 提交申请，等待教师审核

**查看进度：** 在「我的证明」列表点击任一项查看详情和办理进度。

**重新提交：** 如果申请被退回，点击进入详情，修改后重新提交。

**下载证明：** 审核通过并发放后，在详情页可下载 PDF 文件。

### 3.6 智能问答

1. 点击底部「**智能问答**」Tab
2. 可以直接点击常见问题快速提问
3. 也可以在底部输入框输入问题
4. 助手会根据关键词自动匹配答案

支持的问题类型：入党流程、推优条件、证明办理、请假流程、奖学金、成绩单、密码修改等。

### 3.7 个人中心

1. 点击底部「**我的**」Tab
2. 查看个人信息：学号、班级、年级、专业、政治面貌等
3. **更换头像**：点击头像 → 从相册选择或拍照
4. **修改密码**：点击「修改密码」→ 输入旧密码和新密码 → 确定
5. **退出登录**：点击「退出登录」

## 四、教师 / 管理员使用指南（Web 管理端）

### 4.1 登录

浏览器打开管理端地址 → 输入用户名和密码 → 登录

默认管理员账号：`admin` / `admin123`

### 4.2 首页 Dashboard

登录后首页按角色显示功能快捷入口，点击卡片直接进入对应功能。

### 4.3 学生进度总览

查看所有学生的党团流程状态。支持按年级、班级、政治面貌筛选。

### 4.4 流程模板管理

创建和编辑党团流程模板：

1. 点击「流程模板管理」
2. 点击「新建流程」→ 填写流程名称、类型（党建/团建）、描述
3. 配置流程节点：每个节点的名称、审核人角色、所需材料

### 4.5 材料审核

1. 点击「材料审核」
2. 筛选待审核状态的材料
3. 点击「查看」查看材料详情
4. 点击「通过」或「退回」（退回需填写原因）

> 学生提交材料后，对应角色的教师会收到待办通知。

### 4.6 电子证明管理

1. 点击「电子证明管理」
2. 筛选待审核的证明申请
3. 点击「查看」进入详情
4. 审核操作：
   - **通过**：申请进入已通过状态
   - **退回**：填写退回原因，学生可修改后重新提交
5. **办理发放**：已通过的申请，填写 PDF 文件地址后点击「确认发放」

### 4.7 统计分析报表

1. 点击「统计分析报表」
2. 顶部概览卡片：学生总数、材料总数、待审核数、今日上传数等
3. 下方三个统计表：材料状态分布、流程状态分布、证明申请统计
4. 底部 7 天文件上传趋势表
5. 每个统计表右上角可点击「导出」下载 CSV

### 4.8 文件管理

1. 点击「文件管理」
2. 浏览所有上传的文件，支持按关联类型、关键词筛选
3. 操作：
   - **预览**：在线查看文件
   - **下载**：下载到本地
   - **删除**：软删除文件

### 4.9 用户管理（仅管理员）

1. 点击「用户管理」
2. **创建用户**：点击「新建用户」→ 填写信息 → 确定
3. **编辑用户**：修改姓名、手机号、邮箱等
4. **重置密码**：为用户设置新密码
5. **启用/禁用**：控制用户登录权限

### 4.10 系统日志（仅管理员）

查看所有用户的操作记录：登录、创建用户、审核材料、上传文件等。

## 五、常见问题

**Q：忘记密码怎么办？**
A：联系管理员或辅导员重置密码。

**Q：小程序登录显示网络异常？**
A：检查手机网络连接，或联系管理员确认服务器状态。

**Q：材料审核没有我要审核的？**
A：确认筛选条件（状态选择「待审核」），确认你的角色与流程节点要求的审核角色匹配。

**Q：证明申请被退回后怎么处理？**
A：进入「我的证明」→ 点击被退回的申请 → 查看退回原因 → 修改后点击「重新提交」。

**Q：上传文件有什么限制？**
A：支持 pdf、doc、docx、xls、xlsx、png、jpg、jpeg、txt 格式，单个文件不超过 20MB。

**Q：怎么下载已发放的证明？**
A：小程序「我的证明」→ 点击已发放的证明 → 详情页底部有 PDF 下载入口。
