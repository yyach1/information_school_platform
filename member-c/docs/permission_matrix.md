# 权限矩阵（初稿）

## 角色定义

| 角色 | 标识 | 说明 |
| --- | --- | --- |
| 学生 | STUDENT | 仅访问本人相关流程、材料、证明、通知与问答 |
| 教师/审核人员 | TEACHER | 访问所负责范围内学生数据，执行审核与统计 |
| 系统管理员 | ADMIN | 全局管理与安全审计，拥有系统配置与日志访问权限 |

## 能力矩阵

| 能力/模块 | 说明 | STUDENT | TEACHER | ADMIN |
| --- | --- | --- | --- | --- |
| 登录认证 | 登录、获取当前用户信息、退出/刷新 | ✅ | ✅ | ✅ |
| 用户资料 | 查看本人资料、修改本人联系方式等 | ✅ | ✅ | ✅ |
| 用户管理 | 创建/禁用/重置密码/修改角色 | ❌ | ❌ | ✅ |
| 通知列表 | 分页查询本人通知/待办 | ✅ | ✅ | ✅ |
| 未读数 | 查询本人未读通知数量 | ✅ | ✅ | ✅ |
| 标记已读 | 标记本人通知已读 | ✅ | ✅ | ✅ |
| 通知生成（业务联动） | 提交材料/审核通过退回/证明进度变更触发通知 | ✅（被动接收） | ✅（被动接收） | ✅（被动接收） |
| 通知生成（系统） | 系统内部服务创建通知（供流程/审批模块调用） | ❌ | ❌ | ✅ |
| 学生进度总览（管理端） | 按年级/班级/状态筛选与查看学生进度 | ❌ | ✅ | ✅ |
| 系统日志查看 | 查询关键操作日志（分页/筛选） | ❌ | ❌ | ✅ |
| 日志记录 | 关键接口自动记录操作日志 | ✅（被记录） | ✅（被记录） | ✅（被记录） |

## 接口级约束（建议）

| 类别 | 规则 |
| --- | --- |
| 鉴权 | 除登录接口外，所有接口必须携带有效身份凭证 |
| 越权 | STUDENT 仅允许访问 user_id = 本人 的资源；TEACHER 仅允许访问其负责范围内资源；ADMIN 全局 |
| 审计 | 登录、上传、提交、审核、导出、模板修改等写入 operation_log |
